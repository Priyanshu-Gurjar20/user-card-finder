# 🔍 User Card Finder

A clean and interactive **User Search App** that displays stylish profile cards and filters them in real-time as you type.

🚀 Built using **HTML, CSS, JavaScript, and Tailwind CSS**

## 📸 Preview

A minimal UI that shows user cards with:

* Background image
* Blur overlay effect
* Name & bio
* Live filtering search

## ⚙️ How It Works

* User data is stored in a JavaScript array
* Cards are dynamically created using DOM methods
* Search input filters users using string matching
* UI updates instantly without reload
